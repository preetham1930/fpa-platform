# Project: FP&A Platform (Google Corp Eng portfolio build)
Monorepo: backend/ (FastAPI), frontend/ (React+Vite), integration/, pipeline/, docs/.
Stack: FastAPI + SQLAlchemy + Pydantic, Postgres (local Docker now, Cloud SQL later),
Recharts frontend, Apache Beam pipeline -> BigQuery, deployed on Google Cloud Run.
Domain: financial planning & analysis (budgets, actuals, variance, forecasting).
